# Portafolio Profesional - Jonathan Sonda

Este es mi proyecto de portafolio personal desarrollado durante el Sprint de Desarrollo Web. Es un sitio web responsivo diseñado para mostrar mi perfil como Desarrollador Full-Stack Junior, mis habilidades técnicas y los proyectos en los que he trabajado.

## 🚀 Funcionalidad

El sitio funciona como una tarjeta de presentación digital dinámica que incluye:
- **Introducción:** Presentación personal con una estética moderna mediante el uso de degradados radiales.
- **Habilidades y Herramientas:** Una sección centrada que agrupa los iconos de las tecnologías que domino, utilizando una disposición flexible que se adapta a pantallas móviles.
- **Proyectos Destacados:** Galería de proyectos con tarjetas que incluyen un "velo" de gradiente para mejorar la legibilidad de las tecnologías utilizadas.
- **Diseño Responsivo:** Adaptación completa para resoluciones desde escritorio hasta móviles pequeños (320px).
- **Formulario y Contacto:** Enlaces directos a mis redes profesionales y métodos de contacto.

## 🛠️ Tecnologías y Técnicas Utilizadas

Para este proyecto utilicé las siguientes herramientas y metodologías:

- **HTML5 Semántico:** Estructura clara y accesible del contenido.
- **CSS3 Avanzado:** - **Flexbox:** Para la alineación y distribución del diseño responsivo.
  - **Metodología BEM (Block Element Modifier):** Para mantener un código CSS organizado, escalable y fácil de leer.
  - **Media Queries:** Ajustes precisos para múltiples resoluciones (Escritorio, Tablet, Mobile).
  - **Efectos Visuales:** Uso de `linear-gradient` y `radial-gradient` para crear profundidad y mejorar el contraste visual.
- **Git & GitHub:** Control de versiones y flujo de trabajo profesional mediante ramas y commits.
